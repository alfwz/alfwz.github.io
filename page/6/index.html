LC020 Valid Parentheses

Posted on 06-06-2020 In Leetcode , Stack Disqus:

Description

Example 1:

Input: “()”
Output: true
Example 2:

Input: “()[]{}”
Output: true
Example 3:

Input: “(]”
Output: false
Example 4:

Input: “([)]”
Output: false
Example 5:

Input: “{[]}”
Output: true

Solution

  • stack

    class Solution {
  Map<Character, Character> mappings =new HashMap<Character, Character>()<!--￼0-->;

  public boolean isValid(String s) {
      Stack<Character> parenthesis =new Stack<>();
      //遍历符号串
      for(int i=0; i<s.length(); i++){
          char c = s.charAt(i);
          //如果这个char是一种左括号 放进stack这个char的右括号
          if(mappings.containsKey(c)){
              parenthesis.push(mappings.get(c));
          }
          //如果这个char是一种右括号
          else if(mappings.containsValue(c)){
              //如果这个stack空了 或者 
              //因为之前放进了对应的右括号pop出来的不是这个有括号则false
              //判断stack为空用isEmpty（）
              if(parenthesis.isEmpty() || parenthesis.pop()!=c){
                  return false;
              }
          }
      }
      return parenthesis.isEmpty();
  }
}
